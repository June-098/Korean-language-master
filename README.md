# 🇰🇷 Korean Language Master

A mobile-first app for learning Korean from scratch — built for absolute beginners and themed around K-pop, Korean food, and Korean culture to make studying fun.

**▶️ Try it live:** https://korean-language-learning.netlify.app/

## What it is

Korean Language Master walks you through the fundamentals of Korean across six themed chapters, each with bite-sized lessons, audio pronunciation, and quizzes to check what you've learned.

- **Chapter 1** — The Korean alphabet (Hangul): consonants, vowels, and syllable blocks
- **Chapter 2** — Basic words: numbers, colors, food, family, and everyday vocabulary
- **Chapter 3** — Greetings & meeting people (formal and informal)
- **Chapter 4** — At the restaurant: ordering and paying
- **Chapter 5** — Shopping & convenience stores
- **Chapter 6** — Going out for drinks & socializing

## Features

- 📖 Clear study material with romanization and English meanings
- 🔊 Tap-to-hear Korean pronunciation
- 📝 A 10-question word quiz and a 10-question fill-in-the-blank quiz per chapter
- 💾 Your progress and quiz scores are saved on your device
- 📱 Runs on iPhone, Android, and any web browser

## Built with

Expo · React Native · React Navigation · TypeScript
